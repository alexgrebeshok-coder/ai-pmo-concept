# AI-PMO for Builders

**Универсальный фреймворк для автоматизации управления строительными и инвестиционными проектами**

## Концепция

AI-PMO for Builders — это набор специализированных скиллов для Claude Code, которые работают вместе как виртуальный проектный офис (PMO). Система автоматизирует полный цикл управления проектом: от планирования до отчётности генеральному директору.

Применим для любой строительной, девелоперской или проектной компании.

## Что это даёт

- **Автоматическое планирование** — графики Ганта, распределение ресурсов, критический путь
- **Контроль выполнения** — план-факт анализ, расчёт индексов SPI/CPI (Earned Value Management)
- **Управление рисками** — идентификация, оценка, матрица рисков, стратегии митигации
- **Отчётность** — еженедельные дашборды для руководства, автоматические алерты
- **Документооборот** — письма заказчику, обработка актов КС-2/КС-3, анализ договоров
- **Сбор данных** — электронный журнал работ через Telegram-бот

## Скиллы системы

| Категория | Скиллы |
|-----------|--------|
| **Планирование** | Графики Ганта, Распределение ресурсов |
| **Контроль** | План-факт анализ (EVM), Электронный журнал работ |
| **Аналитика** | Риски, Прогнозирование, Алерты об отклонениях |
| **Документы** | Письма заказчику, Обработка актов, Анализ договоров |
| **Отчётность** | Дашборды для руководства |

## Особенности

### Методологии
- **EVM** (Earned Value Management) — SPI, CPI, EAC, ETC
- **Матрица рисков** 5×5 с автоматическими порогами
- **Критический путь** (CPM) для графиков

### Российские стандарты
- Формы КС-2, КС-3, КС-6а
- ГОСТ Р 7.0.97-2016 для деловых писем
- Интеграция с 1С:PM и 1С:ERP

### Адаптивность
- Настраивается под любую компанию
- Поддержка различных климатических зон (включая Крайний Север)
- Масштабируется от одного проекта до портфеля

## Выходные форматы

- **Excel** — графики, матрицы, план-факт, бюджеты
- **Word** — письма, отчёты, акты
- **Telegram** — оперативные алерты и сбор данных
